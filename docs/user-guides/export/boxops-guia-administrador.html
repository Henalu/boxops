Gestión práctica de centros, equipo, catálogo, plantillas, horario y cobertura.

Guía rápida para trabajar con BoxOps sin perder contexto.

Para quién es esta guía

Para quien configura la operativa diaria y mantiene los datos listos para trabajar.

Qué puedes hacer

  • Crear centros y tipos de actividad.
  • Crear o invitar usuarios y mantener fichas operativas.
  • Crear plantillas semanales y aplicarlas a semanas concretas.
  • Editar bloques del horario y asignar entrenadores.
  • Revisar cobertura, solicitudes, ausencias y fichajes según permisos activos.

Uso recomendado

  • Empieza por las pantallas principales de tu rol.
  • Si una acción no aparece, puede depender de tus permisos.
  • No des por hecho que un botón “debería estar”: si no aparece, puede ser por rol.
  • Reporta página, acción, resultado esperado y resultado real.

Flujos principales

Empieza por estos recorridos. Son los que más valor dan en el uso diario.

Dar de alta a una persona

  1. Entra en Equipo.
  2. Elige Invitar usuario si quieres que acepte por email.
  3. Elige Crear cuenta si vas a darle una contraseña temporal.
  4. Asigna rol, estado y ficha operativa.
  5. Comprueba que queda como cuenta vinculada y operativa.

Crear un centro

  1. Entra en Centros.
  2. Crea el centro con nombre, slug y zona horaria.
  3. Mantén el centro activo si se va a usar en horario.
  4. Desactívalo, no lo borres, si deja de usarse.

Crear el catálogo de clases

  1. Entra en Tipos de actividad.
  2. Crea cada actividad con nombre, categoría, color y entrenadores necesarios.
  3. Usa 0 entrenadores solo para bloques que no necesitan cobertura.
  4. Desactiva tipos antiguos en vez de eliminarlos.

Editar el horario

  1. Abre Horario y elige semana y centro.
  2. Crea bloque manual con el botón + si es algo puntual.
  3. Abre un bloque para cambiar hora, tipo, notas o asignaciones.
  4. Cancela un bloque si no se hará; no lo borres de la historia operativa.

Límites actuales

  • No uses el rol de administrador para acceder a datos sensibles que no estén diseñados para ese permiso.
  • No prometas automatismos: cobertura, ausencias y eventos siguen requiriendo revisión humana.
  • No cargues datos reales sensibles hasta cerrar entorno, permisos y validación.

Si necesitas reportar algo

  1. Di qué estabas intentando hacer.
  2. Indica la pantalla: por ejemplo, Horario, Equipo o Mi fichaje.
  3. Explica qué esperabas que pasara.
  4. Explica qué pasó realmente.
  5. Añade captura si el problema es visual o de comprensión.

Evita enviar contraseñas, datos sensibles o información laboral real por canales no acordados.
