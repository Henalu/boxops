# Guias de usuario - BoxOps

Guias practicas por rol. Estan incompletas a proposito: el producto ya tiene cobertura basica, plantillas semanales y dashboard admin basico, pero aun no tiene dashboard visual final, cambios, ausencias, fichaje ni documentos.

## Roles actuales

- `admin`: puede entrar, resolver organizacion, ver dashboard basico de cobertura y gestionar centros, memberships, perfiles basicos de coach, tipos de actividad, bloques operativos semanales, asignaciones, filtros, cobertura basica y plantillas semanales.
- `coach`: puede entrar, resolver organizacion y consultar centros, coaches, tipos de actividad, bloques operativos, asignaciones, cobertura, "Mi horario" y plantillas en modo lectura.

## Guias

- `admin.md`: uso actual para admin.
- `coach.md`: uso actual para coach.
